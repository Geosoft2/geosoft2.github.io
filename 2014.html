Ge(c)omment

Open Source code from Geosoft II seminar at ifgi, winter semester 2014/15

Organisation page

During the winter semester 2014/15 the bachelor students at the Institute for Geoinformatics (ifgi) worked on a web platform for commenting on geospatial datasets.
The course Geosoft II introduces students to advanced application development (web services, user interfaces) for geospatial data. This year the students had to work with standardized metadata for geospatial data (WMS, WFS, ...) and create a user-friendly platform to create new geospatial data in the form of user comments on spatial data.
The course was taught by Holger Fritze and Daniel NÃ¼st.

In the following we will shortly present the results of the five groups - be sure to try out the demos (from within university network only). And of course all source code is open source and available on GitHub: https://github.com/organizations/Geosoft2/

CommenTerra

CommentTerra is a web application, which allows a community-based commenting, rating, sharing, and searching of geospatial data sets. Users should be encouraged to engage in a discussion though their reviews.

Demo: http://giv-geosoft2a.uni-muenster.de/
Code: https://github.com/ZoeEsaulova/CommenTerra

My Meta Maps

My Meta Maps is a web directory for easy discovery of geodata based on aggregated metadata and user reviews.

Demo: http://giv-geosoft2b.uni-muenster.de
Code: https://github.com/m-mohr/my-meta-maps

Cartostoph!

Cartostroph! is a web application for commenting geodata.

Live Demo: http://giv-geosoft2c.uni-muenster.de
Code: https://github.com/mruhe/cartostroph

Zakbum

Zakbum is a web application, where anonymous and registered users alike can comment on and rate geospatial datasets. A search function with multiple filtering options allows to discover datasets.

Link: http://giv-geosoft2d.uni-muenster.de
Code: https://github.com/l-bode05/Geosoft2-Gruppe4

georate

georate is a platform for commenting and rating geodata. Users can comment themselves as well as explore existing reviews.

Link: http://giv-geosoft2e.uni-muenster.de
Code: https://github.com/mdragunski/Geosoft2-WS14-15-Gruppe5
